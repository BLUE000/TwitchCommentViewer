# TwitchCommentManager

TwitchCommentManagerは、Twitch配信者向けの多機能コメント管理ツールです。
配信中のコメントをリアルタイムで取得し、OBSなどの配信画面への表示や「棒読みちゃん」による音声読み上げ、さらにスパムコメントの自動判定や過去ログの閲覧などを一括で行うことができます。

プログラムの知識がない方でも、簡単なボタン操作だけで設定できるように設計されています。

---

## 🚀 主な機能

1. **Twitchコメントのリアルタイム取得**
   - 難しい設定不要で、ボタン1つであなたのTwitchアカウントと連携できます。
2. **OBS連携（画面へのコメント表示）**
   - 専用のローカルWebサーバーを内蔵しており、短いURLをOBSに貼り付けるだけでコメントを配信画面に表示できます。
   - 自作のデザイン（オーバーレイHTML）を作って自由に着せ替えることも可能です。
3. **棒読みちゃん連携（音声読み上げ）**
   - コメントが投稿された瞬間に、自動的に棒読みちゃんにテキストを送信して読み上げさせます。
4. **コメントの自動分析・スパム対策**
   - 荒らしコメントやスパムを自動的に判定し、配信画面に表示させないなどの安全対策機能を搭載しています。
5. **過去ログ閲覧ツール（DB Viewer）**
   - 配信中のコメントやスパム判定の結果はすべてデータベースに記録され、付属の専用ビューワからいつでも表形式で振り返ることができます。

## 📥 使い方

### 1. 起動とTwitchアカウントの連携
1. ダウンロードしたフォルダの中にある `TwitchCommentManager.exe` をダブルクリックして起動します。
2. アプリが起動したら「設定」タブを開きます。
3. **「Twitch 連携を開始 (OAuth認証)」** というボタンを押します。
4. ブラウザが自動的に開き、Twitchのログイン・連携確認画面が出ますので「許可」を押してください。
5. 認証が完了すると、自動的にあなたの配信枠に接続され、コメントの取得が開始されます。

### 2. 配信画面（OBS）へのコメント表示
1. 「設定」タブの **「▼ OBS連携設定」** をクリックして開きます。
2. 「HTTPサーバーポート」は基本的にはそのまま（`8081`など）で大丈夫です。
3. **「URLコピー」** ボタンを押します。
4. OBSを開き、「ソース」の「＋」ボタンから **「ブラウザ」** を追加します。
5. プロパティ画面が開いたら、「URL」の欄に先ほどコピーしたURL（例: `http://localhost:8081/`）を貼り付けます。
6. 以上で、TwitchのコメントがOBS上にリアルタイムで表示されるようになります！

**🎨 デザイン（オーバーレイ）を変更したい場合**
- 設定画面の「フォルダを開く」ボタンを押すと、オーバーレイファイルが入っているフォルダが開きます。
- この中に自分で作ったHTMLファイルを入れると、設定画面の「オーバーレイファイル」のプルダウンから選べるようになります。

### 3. 棒読みちゃんでコメントを読み上げる
1. あらかじめパソコンで「棒読みちゃん」を起動しておきます。
2. TwitchCommentManagerの「設定」タブから **「▼ 棒読みちゃん連携設定」** を開きます。
3. 「実行ファイルパス」の **「参照...」** ボタンを押し、棒読みちゃんの本体（`BouyomiChan.exe`）を選択します。
4. **「テスト送信」** ボタンを押して、棒読みちゃんが「テスト音声です」と喋れば連携成功です！
5. **「保存」** ボタンを押しておきましょう。

### 4. 過去のコメントログを見返す
配信中に流れたコメントは自動的に記録されています。
ログを見たい時は、アプリと同じフォルダに入っている **`TwitchCommentDbViewer.exe`** をダブルクリックして起動してください。
いつ、誰が、どんなコメントをしたのか、それがスパム判定されたのかどうかが一覧で確認できます。
