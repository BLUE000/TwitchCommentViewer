# TwitchCommentViewer

TwitchCommentViewer、Twitchで配信を行うストリーマー（配信者）のための、やさしくて多機能なコメント管理・管理サポートツールです。

「配信中のコメントを画面に出したい」「コメントを声で読み上げさせたい」「モデレーターとしての管理をもっと楽にしたい」「荒らしやスパム対策を自動化したい」といった要望を、プログラムの専門知識がない初心者の方でもボタン操作だけで簡単に実現できます。

---

## 🚀 主な機能

1. **簡単Twitch連携**：ボタンを1回押してブラウザで許可するだけで、すぐにコメントの取得が始まります。
2. **チャットのピン留め（固定表示）**：コメント欄で右クリックするだけで、大事なコメントを配信画面やTwitchチャットの最上部に固定表示できます。
3. **アナウンス送信**：コメントタブの下部から、背景色付きの目立つ「アナウンスチャット」を直接送信できます。
4. **OBS連携（コメント表示）**：OBSにコピーしたURLを貼り付けるだけで、配信画面にコメントをおしゃれに表示できます。
5. **音声読み上げ（棒読みちゃん / VOICEVOX連携）**：流れてきたコメントを音声合成エンジンで自動的に読み上げさせます。音量・速度・音程をエンジン個別に設定可能です。
6. **視聴者管理＆モデレーション**：視聴者リストからワンクリックで、シャウトアウト（紹介）、VIP追加/削除、モデレーター任命/解除、タイムアウト（一時的な発言禁止）、BAN（追放）が行えます。
7. **コメント自動解析＆集計**：コメントの感情（ポジティブ・ネガティブなど）の分析や、スパム検知、コメント数の推移をグラフでリアルタイム表示します。
8. **ログ見返しツール（DB Viewer）**：過去の配信コメントをいつでも表形式で見返すことができます。

## 📥 セットアップと使い方

### 1. 起動とTwitchアカウントの連携
1. ダウンロードしたフォルダの中にある `TwitchCommentManager.exe` をダブルクリックして起動します。
2. 画面の右側にある **「設定」** タブを開きます。
3. **「Twitch 連携を開始 (OAuth認証)」** という青いボタンを押します。
4. インターネットブラウザが自動的に開き、Twitchのログイン・連携確認画面が表示されます。画面の指示に従って **「許可」** ボタンを押してください。
5. 認証が完了するとブラウザに「認証に成功しました」と表示され、アプリの下部ステータスバーに「自動接続を開始します」と表示されます。これで準備完了です！

> [!IMPORTANT]
> **旧バージョンからアップデートした方へ**
> ピン留め機能やアナウンス機能などの新しい機能を利用するには、追加のアクセス権限が必要です。アップデート後は一度「設定」タブを開き、再度 **「Twitch 連携を開始 (OAuth認証)」** ボタンを押して連携のやり直し（再認可）を行ってください。

---

### 2. 音声でコメントを読み上げる（棒読みちゃん / VOICEVOX）
流れてきたチャットコメントを音声で読み上げさせたい場合の設定手順です。「棒読みちゃん」または「VOICEVOX」のいずれかの音声合成エンジンと連携できます。

#### A. 棒読みちゃんを使用する場合
1. あらかじめパソコンで「棒読みちゃん」アプリを起動しておきます。
2. 本ツールの **「設定」** タブを開き、 **「▼ 読み上げ設定」** をクリックして展開します。
3. **「棒読みちゃんを使用する」** のラジオボタンを選択します。
4. 「実行ファイルパス」の右側にある **「参照...」** ボタンを押し、お使いのパソコン内にある棒読みちゃんの本体ファイル（`BouyomiChan.exe`）を選択します。
5. 音量、速度（%）、音程（%）をお好みに合わせて調整します。（※音量 `-1` は「棒読みちゃん本体側の音量をそのまま使用する」設定になります）
6. 調整またはエンジン選択を行うと、右下の保存ボタンが **「設定を保存 (要保存*)」** とオレンジ色で強調表示されるので、クリックして保存します。
7. 「テスト送信」欄に文字を入力して **「テスト送信」** ボタンを押し、棒読みちゃんから音声が聞こえれば接続成功です！

#### B. VOICEVOXを使用する場合
1. あらかじめパソコンで「VOICEVOX」アプリを起動しておきます。
2. 本ツールの **「設定」** タブを開き、 **「▼ 読み上げ設定」** をクリックして展開します。
3. **「VOICEVOXを使用する」** のラジオボタンを選択します。
4. 「実行ファイルパス」の右側にある **「参照...」** ボタンを押し、お使いのパソコン内にある VOICEVOX の本体ファイル（`VOICEVOX.exe`）を選択します。
5. 使用したいキャラクターの「話者ID」（初期値: `3` [ずんだもん] など）を入力します。
6. 音量、速度（倍率）、音程をお好みに合わせて調整します。
7. 右下の **「設定を保存 (要保存*)」** ボタンをクリックして保存します。保存すると、動的にVOICEVOXとの接続が開始されます。
8. 「テスト送信」欄に文字を入力して **「テスト送信」** ボタンを押し、VOICEVOXから音声が聞こえれば接続成功です！

> [!TIP]
> **連動起動・自動終了について**
> 「起動時に読み上げツールを自動起動する」「終了時に読み上げツールを自動終了する」にチェックを入れて設定を保存すると、本ツールの起動・終了に合わせて棒読みちゃんやVOICEVOXのプロセスが自動で連動して起動・終了します。

#### 💡 読み上げを無視する（除外設定）
「ボットの自動投稿や、特定のユーザーのコメントは読み上げさせたくない」という場合は、以下の設定を行ってください。
* **ボットの自動除外**：設定タブの「▼ BOT判定・除外設定」にある **「BOTユーザー（カンマ区切り）」** に登録されたユーザー（例: `Nightbot` など）のコメントは、自動的に読み上げから除外されます。
* **特定のユーザーの除外**：設定タブの「▼ 読み上げ設定」にある **「除外ユーザー (カンマ区切り)」** の欄に、読み上げをスキップしたいTwitchのユーザーID（半角小文字）をカンマ（`,`）で区切って入力し、保存ボタンをクリックします。

---

### 3. 配信画面（OBS）へコメントを表示する
配信画面（OBS Studioなど）にリアルタイムでコメントを流すための設定です。

1. **「設定」** タブを開き、 **「▼ OBS連携設定」** をクリックします。
2. 画面上部にある **「OBSファイル出力を有効化」** および **「OBS WebSocketを有効化」** スイッチを **ON** にします。
3. 右側にある **「URLコピー」** ボタンを押します。（クリップボードに `http://localhost:8081/` のようなアドレスがコピーされます）
4. **OBS Studio** を起動します。
5. OBSの「ソース」エリアにある **「＋」** ボタンを押し、 **「ブラウザ」** を選択します。
6. 新規作成の画面が出たら適当な名前（例: 「コメ欄」など）をつけて「OK」を押します。
7. プロパティ画面が表示されるので、 **「URL」** の欄に先ほどコピーしたアドレスを貼り付け（Ctrl + V）、「OK」を押します。
8. これで配信画面にコメントが表示されるようになります！

## 👥 視聴者を管理する（「視聴者」タブの使い方）

「視聴者」タブでは、今チャット欄に接続しているすべてのユーザーを確認し、モデレーション操作を行うことができます。

### 1. リストの確認方法
視聴者は、画面上に並ぶ「アコーディオン（アコーディオンのタイトルボタンを押すと開閉する枠）」ごとに自動で分類されます。複数の役割を持っているユーザーは、それぞれのリストに重複して表示されます（例：Nightbotは「モデレーター」と「チャットボット」の両方に表示されます）。

* **すべて**：現在接続しているすべての視聴者
* **ストリーマー**：配信者本人（あなた）
* **モデレーター**：モデレーター権限（盾マーク 🛡️）を持つユーザー
* **VIP**：VIP権限（ダイヤマーク 💎）を持つユーザー
* **アーティスト**：アーティスト属性を持つユーザー
* **チャットボット**：設定でBOT除外に指定されているボットアカウント
* **一般**：その他の一般視聴者

> [!TIP]
> **API負荷を抑えるための制限（クールダウン）**
> 視聴者リストは「視聴者」タブを開いたときに自動で取得されますが、何度も連続でAPIを呼び出してTwitch側から制限を受けるのを防ぐため、以下の自動制限（クールダウン）を設けています。
> * **自動更新**：10分に1回自動で行われます（他のタブを開いている間は自動取得タイマーが完全に停止し、パソコンの動作が軽くなります）。
> * **手動「更新」ボタン**：前回の取得から3分以上経過するとボタンが青く光って押せるようになります。

### 2. 各種アクションボタン（モデレーション）の使い方
リスト内の視聴者の名前にマウスカーソルを合わせる（ホバーする）、またはクリックして選択すると、名前の右端に操作ボタンが浮かび上がります。
*(※配信者自身の名前には誤操作防止のためボタンは表示されません)*

* **[シャウトアウト]**：その人が配信をしているストリーマーである場合、その人のチャンネル紹介（シャウトアウトコマンド）をチャットに送信します。
  - *安全対策*：連続で使用してスパムになるのを防ぐため、一度押すと**2分間のクールタイム**が発生し、ボタンが一時的に無効になります。クールタイムの残りカウントダウンは本アプリのウィンドウタイトルバーに表示されます。
* **[VIP]**：押すたびに、VIPの付与と剥奪を切り替えます（トグル動作）。
* **[モデレータ]**：押すたびに、モデレーターの任命と解除を切り替えます。
  - *安全対策*：任命・解除を実行する前に「本当に変更しますか？」という確認メッセージが表示されます。
* **[タイムアウト]**：その視聴者を一時的に発言禁止にします。
  - *手順*：ボタンを押すと何秒間発言禁止にするかを尋ねる画面が出るので、任意の秒数を入力します。その後、確認画面で「はい」を選ぶと実行されます。
* **[BAN]**：その視聴者を永久に追放（BAN）します。
  - *安全対策*：誤操作を防ぐため、実行前に「本当にBANしますか？」という大きな警告確認画面が表示されます。

## 📌 コメントを固定（ピン留め）する

Twitch公式の「ピン留め（固定表示）」機能と連携し、注目させたいコメントをチャット欄の最上部に固定できます。

1. **「コメント」** タブを開きます。
2. 流れてきたコメントの一覧から、ピン留めしたいコメントを **右クリック** します。
3. メニューから **「📌 チャット上部にピン留めする」** を選び、固定したい時間（30秒、1分、5分、15分、30分、または配信終了まで）を選択します。
4. Twitchのチャット最上部にコメントが固定表示されます。
5. 固定を途中でやめたい場合は、コメント欄の任意の場所を右クリックし、メニューの **「❌ ピン留めを解除」** をクリックします。

## 📢 アナウンス（チャットに目立たせて送信する）

メッセージの背景に色をつけ、チャット欄で目立たせて送信する機能です。配信の宣伝や重要なお知らせに役立ちます。

1. **「コメント」** タブを開きます。
2. 画面の最下部にあるテキスト入力欄に、流したいメッセージ（最大500文字）を入力します。
3. 左側にあるプルダウンメニューをクリックし、アナウンスの背景色を選択します。
   - **通常**：チャンネルのテーマカラー
   - **青** / **緑** / **オレンジ** / **紫**
4. 右端の **「アナウンス送信」** ボタンを押すか、キーボードの **Enterキー** を押すとチャットにアナウンスが送信されます。

## 📊 解析結果・グラフの見方

配信を盛り上げるためのデータ分析機能です。 **「解析」** タブから確認できます。

### 1. 「解析ログ」サブタブ
コメント解析機能が検知したシステムログがリアルタイムで表示されます。
* **感情スコア**：コメントがポジティブ（肯定・楽しい）かネガティブ（否定・悲しい）かを判定します。
* **トレンドワード**：短時間に多く発言されたキーワード（草、すご、など）を検知してログに記録します。
* **スパム検知**：荒らし行為やスパムと判定された危険なコメントを自動検知して記録します。

### 2. 「コメント集計」サブタブ
配信中の盛り上がりを視覚的なグラフや数値で確認できます。
* **グラフ切り替え**：
  - **総コメント数推移**：配信中のコメントの勢いの変化を折れ線グラフで表示します。
  - **ユーザーごとの推移**：コメント数が多い上位のユーザーごとの発言頻度の推移を色分けされた折れ線グラフで表示します。
* **時間粒度（横軸の細かさ）**：「自動」にしておくと、配信時間に合せてX軸の間隔（5秒〜5分）を自動で綺麗に調整します。
* **表示数（TopN）**：グラフに描画するアクティブユーザーの数を「TOP 3 / TOP 5 / TOP 10 / 全員表示」から選択できます。
  - *パフォーマンス保護機能*：チャット参加者が100人を超えると、描画が重くなるのを避けるために自動で「TOP 10」表示へ制限されます。
* **表（ユーザー統計）**：誰が何回コメントしたかのランキングが自動で更新されます。

## 🎨 オーバーレイの自作・カスタマイズ手順

配信画面に表示されるコメントのデザイン（見た目）を、HTMLとCSSを使って自由に自作・改造することができます。

### 1. カスタマイズ用のフォルダを開く
1. 本ツールの **「設定」** タブを開きます。
2. **「▼ OBS連携設定」** をクリックし、 **「フォルダを開く」** ボタンを押します。
3. コメント表示用のテンプレートがテーマフォルダごとに整理されて格納されたアセットフォルダが開きます。

### 2. デザインの編集方法（CSSの編集）
初心者の方は、まず利用したいテーマフォルダ内（例: `standard/`）の `style.css` をテキストエディタ（メモ帳やVS Codeなど）で開いて編集することをお勧めします。
* **コメントの文字色を変更したい場合**：
  `style.css` 内の `.message` や `.username` の項目を探し、 `color: #ffffff;` のカラーコード（`#` から始まる6桁の英数字）を好みの色に書き換えます。
* **背景に「すりガラス効果（グラスモーフィズム）」や角丸をつけたい場合**：
  `.comment` クラス内の `background-color` を半透明（例: `rgba(0, 0, 0, 0.5)`）にし、`border-radius: 8px;` や `backdrop-filter: blur(5px);` などを追記します。
* **変更の適用**：ファイルを保存した後、OBS上の「ブラウザ」ソースのプロパティ画面で **「現在のページのキャッシュを再読み込み」** ボタンを押すと即座に反映されます。

### 3. 本格的な自作・仕組み（WebSocket接続）
各テーマフォルダ内の `index.html` の中身を書き換えて、表示ロジック自体を改造することも可能です。

本ツールは起動中、内部でWebSocket（通信ポート: **HTTPサーバーポート + 1**。デフォルトでは `8082`）のサーバーを自動で立ち上げています。自作のHTML内に以下のJavaScriptを実装することで、リアルタイムにコメントデータを受け取ることができます。

```javascript
// アプリ起動時に割り当てられたポートに接続する例
const wsPort = window.location.port ? (parseInt(window.location.port) + 1) : 8082;
const ws = new WebSocket('ws://localhost:' + wsPort);

ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    
    // コメント受信時の処理
    if (data.action === 'comment') {
        const username = data.username; // 送信者の名前
        const message = data.message;   // コメント本文
        const avatarUrl = data.avatarUrl; // プロフィール画像のURL
        const badges = data.badges;       // バッジ画像のURLまたは文字列のリスト
        
        // ここでHTMLのDOM操作を行い画面に描画します
    }
};
```
自作した新しいテーマフォルダ（例: `assets/overlay/my_overlay/` 内に `index.html` と `style.css` を配置）を保存すると、設定タブの「オーバーレイファイル」のドロップダウンリストに自動で追加され、選択できるようになります。
